### DialogFlow + Cloud Functions + Firestore

#### Um simples webhook para conectar à uma base de dados

Foi ontem que aconteceu o primeiro evento da série [Build Actions for Your Community](https://developers.google.com/events/buildactions/) em Maputo. Esta série de eventos visa ensinar a desenvolver aplicações utilizando o Google Assistant.

Todos que desenvolverem uma app para o Google Assistant, podem submeter essa app para ser avaliada. E se a app cumprir com [alguns critérios](https://developers.google.com/actions/community/faq#eligibility), os desenvolvedores tornam-se parte da [Comunidade de Desenvolvedores do Google Assistant](https://developers.google.com/actions/community/) onde têm a chance de ganhar vários prêmios.

Com o objetivo de ajudar todos que estiverem interessados em participar da comunidade, decidi escrever este artigo em que mostro como conectar o nosso agente do DialogFlow com uma base de dados ([Cloud Firestore for Firebase](https://medium.com/android-dev-moz/firestore-8d225062ffe8)).
