Firebase News: April 26th — May 09th

This is Article 02 of “Firebase News” — a series of articles where you can find the most recent news from the Firebase World. A new article is posted every 1–2weeks discussing (most of) the changes made on the official tools, SDKs, sample apps and some of the 3rd party libraries. Also expect to see what’s new on: the official Firebase blog, the Firebase Developers Medium publication, the Firebase Youtube Channel and the Firebase Open Source site.

Firebase Admin SDKs

Python Admin SDK

Version 4.2.0 was released and it introduces a newtenant_mgt module that provides APIs to work with Multi-tenancy. You can read more about Multi-tenant applications in this blog post by Hiranya Jayathilaka.

Node.js Admin SDK

Starting in version 8.12.0, developers using the NodeJS Admin SDK can now retrieve and delete users in bulk, using the newly added getUsers() and deleteUsers() functions. Learn more about it on this other blog post by Hiranya.

.Net Admin SDK

Version 1.12.0 of the .Net Admin SDK has 2 new methods to manage Firebase session cookies: CreateSessionCookieAsync() and VerifySessionCookieAsync().

Firebase Open Source

Two (2) new Kotlin projects have been added to firebaseopensource.com :

  • firebase-kotlin-sdk from GitLiveApp. As the name suggests, this project is an unofficial Kotlin SDK for Firebase. Unlike the official Firebase KTX, this project does not extend the Java SDK, instead, it was built on top of it, with Kotlin-first design in mind. It also supports writing for Kotlin multiplatform, which means you can use the same code for Android, iOS and JavaScript.
  • kotfirebase by raipankaj - a Kotlin wrapper for the Firebase SDK. The project aims to bring the usage of each API down to one line of code.

Other Minor Releases

Android

  • Firebase Installations 16.3.0 and Firebase Instance ID 20.1.7 improved the detection of invalid FirebaseOptions values.
  • Crashlytics Gradle plugin version 2.1.0 - minor bug fixes.
  • App Distribution Gradle plugin version 1.4.1 - bug fixes.
  • Analytics 17.4.1 - the Event SELECT_CONTENT has been un-deprecated.

iOS

Version 6.24.0 - bug fixes to Analytics, Authentication, Crashlytics, Realtime Database, Cloud Firestore, In App Messaging, FCM and Remote Config.

Firebase Tools 8.2.0

Version 8.2.0 - Minor improvements and dependency updates.

Firebase Extensions

  • storage-resize-images 0.1.7 - fixed a bug that prevents resized images from being shown in the Firebase Console.
    Fixed an issue where the extensions would resize the wrong image, due to caching errors.
  • firestore-translate-text 0.1.2 - target languages can now be reconfigured.
  • firestore-send-email 0.1.4 - the “default FROM” parameter now accepts a name before the email. E.g: Friendly User <foobar@example.com>
